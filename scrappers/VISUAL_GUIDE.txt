💡 PRO TIPS:
═══════════════════════════════════════════════════════════════════════════════

1. Keep server running                → Start once, test multiple times
2. Don't close Chrome manually        → Auto-closes after scraping
3. Wait for complete scrape           → Takes 20-30 seconds (normal)
4. Check JSON files for full data     → More details than console
5. Use descriptive search terms       → Better results (e.g., "mens-tshirts")

🐛 TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════════════════

Problem                               Solution
────────────────────────────────────────────────────────────────────────────────
"Connection Refused"                  → Start server first (step 1)
"No products found"                   → Try different query, check internet
"ChromeDriver error"                  → Update Chrome browser
Server won't start on port 5000       → Close apps using port 5000
Browser doesn't open                  → Check Chrome is installed
Scraping too slow                     → Normal for Selenium (20-30s)
