┌──────────────────────────────────────────────────────────────────────────────┐
│                           🎯 HOW IT WORKS                                    │
└──────────────────────────────────────────────────────────────────────────────┘

Flow:
    User searches → Frontend → Backend /api/scrape/all
                                    ↓
                          [Scrapes 4 platforms]
                                    ↓
                    Amazon | Flipkart | Reliance | Myntra (Selenium)
                                    ↓
                          Returns all products
                                    ↓
                    Frontend displays 4 columns
                                    ↓
                          User clicks product
                                    ↓
                    Opens in respective website

Myntra Scraping Process:
    1. Launch Chrome (headless mode)
    2. Navigate to Myntra search page
    3. Wait 5 seconds for page load
    4. Scroll down 3 times (load more products)
    5. Extract all product data
    6. Close browser
    7. Return products to frontend

┌──────────────────────────────────────────────────────────────────────────────┐
│                        ⚠️  IMPORTANT NOTES                                   │
└──────────────────────────────────────────────────────────────────────────────┘

✅ First Run:
   - ChromeDriver downloads automatically (one-time, ~10MB)
   - First request may take 40-50 seconds
   - Subsequent requests: 20-30 seconds

✅ Performance:
   - Myntra is slower than other scrapers (Selenium overhead)
   - This is NORMAL and cannot be significantly improved
   - Other platforms (Amazon, Flipkart) remain fast

✅ Resources:
   - Chrome runs in headless mode (no window)
   - Uses ~200-300MB RAM per request
   - CPU usage spikes during scraping

✅ Rate Limiting:
   - Don't make too many Myntra requests quickly
   - Myntra may temporarily block if suspicious
   - Wait 30+ seconds between requests

✅ Errors:
   - If no products found: try different search term
   - If timeout: increase timeout in code
   - If ChromeDriver error: update webdriver-manager

┌──────────────────────────────────────────────────────────────────────────────┐
│                         🐛 TROUBLESHOOTING                                   │
└──────────────────────────────────────────────────────────────────────────────┘

Problem: "ChromeDriver not found"
Solution:
    pip install --upgrade webdriver-manager
    python app.py

Problem: Backend won't start
Solution:
    - Check if port 8000 is available
    - Close other apps using port 8000
    - Try: uvicorn app:app --reload --port 8001

Problem: Frontend won't start
Solution:
    cd frontend
    npm install
    npm run dev

Problem: No Myntra products found
Possible causes:
    ✓ Search query has no results on Myntra (try "mens-tshirts")
    ✓ Myntra blocking automated access (wait & retry)
    ✓ Network issues (check internet)
    ✓ Timeout too short (increase in code)

Problem: Very slow scraping
This is NORMAL!
    - Selenium must render JavaScript
    - 20-30 seconds is expected
    - Cannot be significantly faster

Problem: Frontend not showing Myntra column
Checks:
    ✓ Backend running? (http://localhost:8000)
    ✓ Test API: curl "http://localhost:8000/api/scrape/myntra?query=tshirt"
    ✓ Check browser console (F12) for errors
    ✓ Refresh frontend page
